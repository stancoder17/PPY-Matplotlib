# PPY-Matplotlib
Three tasks to practice Python's matplotlib library 

**Task 1:**
Generate (x, y) data points for two mathematical functions:
- Linear function: y1 = 2x + 1
- Quadratic function: y2 = x² − 3x + 2

Plot both functions on a single chart using matplotlib:
Display the linear function as a scatter plot.
Display the quadratic function as a continuous line.
Add a title, legend, grid, and axis labels.

**Task 2:**
Using matplotlib, create three separate bar charts for the following data — one for each subject — using subplots:
- Plot 1: Mathematics results
- Plot 2: Physics results
- Plot 3: Chemistry results
  
Add a title and axis labels to each subplot.
Arrange all subplots in a single row within one figure, adjusting the figure size appropriately.

Data:
students = ['Anna', 'Bartek', 'Cezary', 'Daria', 'Ela']
mathematics = [78, 88, 95, 72, 80]
physics = [85, 79, 92, 68, 82]
chemistry = [92, 85, 88, 74, 78]

**Task 3:**
Create line plots in matplotlib based on the file Data_Pandas.xlsx to display the number of steps taken by each user throughout the week:
Each line should represent one user.
Use different line styles or colors for each user.
Add the following elements to the chart:
- Chart title: "Number of Steps – FirstName – During the Week"
- Axis labels:
  X-axis: "Day of the Week"
  Y-axis: "Number of Steps"
  
Include a legend identifying the users.
Highlight the points corresponding to each user’s highest step count using the appropriate color and a star marker.
